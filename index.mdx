## Overview

<CardGroup cols={2}>
  <Card title="mihomo Kernel" icon="cpu">
    Built on MetaCubeX/mihomo for stable and reliable proxy networking.
  </Card>
  <Card title="SubStore Support" icon="store">
    Optional SubStore integration for easier subscription management.
  </Card>
  <Card title="Web Panel" icon="monitor">
    Integrated web panel for management and monitoring.
  </Card>
  <Card title="Multi-Architecture" icon="box">
    Supports `armeabi-v7a`, `arm64-v8a`, `x86`, and `x86_64`.
  </Card>
</CardGroup>

## Compatibility

- Android 7.0 and later
- Architectures: `armeabi-v7a`, `arm64-v8a`, `x86`, `x86_64`
- Kernel: `mihomo`
