<Update label="2025-11-27" tags={["里程碑", "测试版"]} description="v0.1.0">
  ## 首个正式版本

  YumeBox 0.1.0 正式发布！本次发布标志着 YumeBox 正式进入新的发展阶段。 项目已从原有架构中完全独立，不再局限于 UI 层面的改动，而是进行了整体的重构。

  ### 当前版本仍处于测试阶段，无法保证所有功能均保持可用与稳定。

  如在使用过程中遇到问题、异常或有改进建议，请在 Issue 中提交反馈

  ### 推广

  如果你觉得好用请帮助我推广：https://github.com/YumeLira/YumeBox

  ### 核心功能

  - 基于 mihomo 内核的代理服务
  - 多种代理协议支持（Shadowsocks、V2Ray、Trojan 等）
  - 灵活的规则分流系统
  - 简洁易用的用户界面

  ### 系统支持

  - Android 7.0 及以上版本
  - 支持 armeabi-v7a、arm64-v8a、x86、x86_64 架构

  ### 特色功能

  - 配置文件 URL 导入
  - 外部控制 API
  - 详细的连接日志
  - 节点延迟测试
</Update>
