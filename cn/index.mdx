## 项目介绍

<CardGroup cols={2}>
  <Card title="Mihomo 内核" icon="cpu">
    基于 MetaCubeX/mihomo 内核，稳定可靠。
  </Card>
  <Card title="SubStore 支持" icon="store">
    可选 SubStore 支持，配置管理更方便。
  </Card>
  <Card title="Web 面板" icon="monitor">
    内置 Web 面板，便于管理和监控。
  </Card>
  <Card title="多架构支持" icon="box">
    支持 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 架构。
  </Card>
</CardGroup>

- Android 7.0 及以上
- 支持 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 架构
- 内核：`mihomo`
